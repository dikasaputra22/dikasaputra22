About me

Overview

I'm Dika a beginner web designer who focuses on UI/UX, layout systems, and creating clean modern interfaces. I enjoy designing pages that feel smooth, readable, and visually balanced without unnecessary clutter.

Right now, I’m practicing color harmony, spacing structure, typography, and consistent design patterns so every section looks connected. I'm also learning how to communicate information clearly using visual hierarchy.

My goal is to create interfaces that look simple but feel premium. I believe every detail matters from spacing, line height, color contrast, down to how the eyes move across the layout.

This portfolio is still improving, but with every update, I’m learning something new. I keep experimenting with darker themes, neon accents, and clean card-style layouts that match my personal design style.
